# Api-tasks-nodejs
Api test Nodejs con Swagger Docs

# Features
- Create, read, update, and delete tasks
- Swagger documentation for API endpoints
- Error handling and validation

# Technologies Used
- Node.js
- Express
- Swagger

# Project Structure
```
/src
    /models
    /routes
index.js
```

# Install Dependencies
```
npm install
```

# Run App Serve
```
npm run dev
```

# Get Documentation
```
/api-docs
```
